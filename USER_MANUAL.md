## 프로그램 소개

**Under Memo**는 자동 저장 기능을 갖춘 현대적인 메모 애플리케이션입니다.

### 주요 특징

- ✅ **자동 저장**: 타이핑 후 500ms마다 자동으로 저장
- 🎨 **풍부한 서식**: 굵게, 기울임, 밑줄, 색상, 하이라이트 등
- 🖼️ **미디어 지원**: 이미지, YouTube/Vimeo 동영상 삽입
- 🎨 **내장 그림판**: 간단한 그림 및 스케치 기능
- 📊 **표 기능**: 셀 편집, 병합, 크기 조절 가능한 표
- 🔍 **검색 기능**: 메모 제목 및 태그 검색
- 📌 **고정 메모**: 중요한 메모를 상단에 고정
- 🔒 **메모 잠금**: 비밀번호로 메모 보호
- 📤 **내보내기**: HTML, Markdown, 텍스트 형식 지원
- 🎨 **파스텔 UI**: 눈이 편안한 파스텔 톤 디자인

### 시스템 요구사항

- **운영체제**: Windows, macOS, Linux
- **Python**: 3.8 이상
- **필수 라이브러리**:
  - customtkinter
  - Pillow (이미지 처리)
  - requests (미디어 썸네일)

## 시작하기

### 설치

```bash
# 필수 라이브러리 설치
pip install customtkinter pillow requests

# 프로그램 실행
python modern_notepad.py
```

### 첫 실행

1. 프로그램이 실행되면 빈 편집기와 좌측 사이드바가 표시됩니다
2. 바로 타이핑을 시작할 수 있습니다
3. 입력 후 500ms 후 자동으로 메모가 저장됩니다
4. 저장된 메모는 좌측 사이드바에 표시됩니다

## 기본 기능

### 메모 작성

1. **새 메모 만들기**
   - 좌측 상단의 `+ 새 메모` 버튼 클릭
   - 또는 빈 공간에서 바로 타이핑 시작

2. **메모 제목**
   - 첫 줄의 처음 20자가 자동으로 제목이 됩니다
   - 제목은 사이드바에 표시됩니다

3. **자동 저장**
   - 타이핑 후 500ms 후 자동 저장
   - 저장 중에는 별도 표시 없음 (백그라운드 저장)

### 메모 관리

#### 메모 목록

좌측 사이드바에 모든 메모가 표시됩니다:

- **🟢 초록색**: 저장 완료된 메모
- **🟣 보라색**: 현재 선택된 메모
- **🔴 빨간색**: 수정 중인 메모 (아직 저장 안 됨)

#### 메모 색상 의미

| 색상 | 상태 | 설명 |
|------|------|------|
| 초록색 (파스텔) | 저장 완료 | 선택되지 않은 일반 메모 |
| 보라색 (파스텔) | 선택됨 | 현재 편집 중인 저장된 메모 |
| 빨간색 (파스텔) | 수정 중 | 변경사항이 아직 저장 안 됨 |

#### 메모 작업

- **메모 선택**: 사이드바에서 메모 클릭
- **메모 제목 변경**: 메모 더블클릭 → 새 제목 입력
- **메모 삭제**: 메모 선택 → 상단 `삭제` 버튼
- **메모 드래그**: 메모를 드래그하여 순서 변경

## 텍스트 서식

### 기본 서식

텍스트를 선택한 후 다음 버튼을 클릭:

| 버튼 | 기능 | 단축키 (Windows/Linux) | 단축키 (Mac) |
|------|------|----------------------|-------------|
| **B** | 굵게 | Ctrl + B | Cmd + B |
| *I* | 기울임 | Ctrl + I | Cmd + I |
| <u>U</u> | 밑줄 | Ctrl + U | Cmd + U |
| ~~S~~ | 취소선 | - | - |

### 색상 및 하이라이트

- **텍스트 색상**: `A▼` 버튼 → 색상 선택
- **배경 하이라이트**: `🎨` 버튼 → 배경색 선택

### 정렬

- **왼쪽 정렬**: `≡` 버튼
- **가운데 정렬**: `≡` 버튼
- **오른쪽 정렬**: `≡` 버튼

### 글꼴 설정

1. **글꼴 변경**
   - 상단 드롭다운에서 글꼴 선택
   - 지원: Arial, Times New Roman, Courier New, 맑은 고딕 등

2. **글꼴 크기**
   - 상단 드롭다운에서 크기 선택
   - 범위: 10pt ~ 28pt

### 서식 페인터

복사한 서식을 다른 텍스트에 적용:

1. 서식을 복사할 텍스트 선택
2. `🖌️` (서식 복사) 버튼 클릭
3. 적용할 텍스트 선택
4. 자동으로 서식 적용됨

## 삽입 기능

### 항목 삽입

#### 글머리 기호 (•)

- 상단 `•` 버튼 클릭
- 현재 줄에 글머리 기호 추가

#### 링크 🔗

1. 링크로 만들 텍스트 선택
2. `🔗` 버튼 클릭
3. URL 입력 (예: https://example.com)
4. 링크를 클릭하면 브라우저에서 열림

### 미디어 삽입

#### 이미지 🖼️

1. `🖼️` 버튼 클릭
2. 이미지 파일 선택 (PNG, JPG, GIF, BMP)
3. 이미지가 메모에 삽입됨

**이미지 조작:**
- **크기 조절**: 이미지 더블클릭 → 너비/높이 입력
- **원본 크기 복원**: 크기 조절 대화상자에서 "원본 크기 복원"
- **삭제**: 이미지 선택 후 Delete 키

#### 동영상 📹

YouTube 또는 Vimeo 동영상 삽입:

1. `📹` 버튼 클릭
2. 플랫폼 선택 (YouTube/Vimeo)
3. 동영상 URL 입력
4. 썸네일이 메모에 삽입됨

**동영상 재생:**
- **단일 클릭**: 선택
- **더블 클릭**: 프로그램 내에서 재생
- **크기 조절**: 더블클릭 후 크기 입력

**지원 플랫폼:**
- YouTube: `https://www.youtube.com/watch?v=VIDEO_ID`
- Vimeo: `https://vimeo.com/VIDEO_ID`

### 그림판 🎨

내장 그림판으로 간단한 그림 그리기:

1. `🎨` 버튼 클릭
2. 캔버스 크기 입력 (기본: 600x400)
3. 그림판이 열림

**그림판 도구:**
- **펜 도구**: 자유롭게 그리기
- **도형**: 선, 직사각형, 타원
- **색상 선택**: 펜 색상 변경
- **펜 굵기**: 1-10px
- **레이어**: 여러 레이어 관리
- **지우개**: 그린 내용 지우기
- **실행 취소/다시 실행**: Ctrl+Z / Ctrl+Y

**완료:**
- `✓ 완료` 버튼 클릭하면 메모에 삽입
- 더블클릭하면 다시 편집 가능

### 표 📊

워드프로세서 스타일의 표 삽입:

1. `⊞` 버튼 클릭
2. 행x열 입력 (예: 3x4)
3. 표가 삽입됨

**표 편집:**
- **셀 입력**: 셀 클릭 후 텍스트 입력
- **셀 선택**: Ctrl+클릭으로 다중 선택
- **우클릭 메뉴**:
  - 행 삽입 (위/아래)
  - 행 삭제
  - 열 삽입 (왼쪽/오른쪽)
  - 열 삭제
  - 셀 병합

**크기 조절:**
- **개별 셀**: 셀 오른쪽/하단 경계선 드래그
- **전체 표**: 우하단 파스텔 블루 핸들 드래그

### 체크리스트 ☑️

할 일 목록 삽입:

1. `☑` 버튼 클릭
2. 체크박스가 삽입됨
3. 클릭하여 완료/미완료 토글

## 메모 관리

### 검색

1. 상단 검색창에 키워드 입력
2. 제목 또는 태그에 포함된 메모만 표시
3. 검색창을 비우면 전체 메모 표시

### 태그

메모에 태그를 추가하여 분류:

1. **태그 추가**
   - 상단 `# 태그` 버튼 클릭
   - 태그 이름 입력 (예: 업무, 개인, 아이디어)
   - Enter로 추가

2. **태그 관리**
   - 상단 `⚙️ 태그 관리` 버튼
   - 태그 목록에서 삭제 가능

3. **태그 검색**
   - 검색창에 태그 이름 입력
   - 해당 태그가 있는 메모만 표시

### 고정 메모 📌

중요한 메모를 상단에 고정:

1. **메모 고정**
   - 메모 우클릭 → "고정/고정 해제" 선택
   - 고정된 메모는 ⭐ 표시

2. **고정 메모만 보기**
   - 상단 `⭐` 버튼 클릭
   - 고정된 메모만 필터링
   - 다시 클릭하면 전체 보기

3. **순서 변경**
   - 고정된 메모끼리 드래그하여 순서 변경

### 메모 잠금 🔒

민감한 내용을 비밀번호로 보호:

1. **잠금 설정**
   - 메모 선택
   - 상단 `🔒 잠금` 버튼
   - 비밀번호 입력 (두 번)

2. **잠긴 메모 열기**
   - 잠긴 메모 클릭
   - 비밀번호 입력
   - 올바른 비밀번호 입력 시 열림

3. **잠금 해제**
   - 메모 열기
   - 다시 `🔒 잠금` 버튼
   - 잠금 해제됨

## 고급 기능

### 내보내기 📤

메모를 다양한 형식으로 내보내기:

1. **내보내기 시작**
   - 메모 선택
   - 상단 `📤 내보내기` 버튼

2. **형식 선택**
   - **HTML**: 서식 포함, 웹 브라우저에서 보기
   - **Markdown**: 마크다운 형식 (GitHub 호환)
   - **텍스트**: 순수 텍스트 (서식 제거)

3. **저장 위치**
   - 파일 저장 대화상자에서 위치 선택
   - 파일 저장

### 찾기 및 바꾸기 🔍

텍스트 검색 및 치환:

1. **찾기**
   - `Ctrl+F` (Windows/Linux) 또는 `Cmd+F` (Mac)
   - 검색어 입력
   - "다음 찾기" 또는 Enter

2. **바꾸기**
   - 찾기 대화상자에서 "바꾸기" 탭
   - 찾을 텍스트와 바꿀 텍스트 입력
   - "바꾸기" 또는 "모두 바꾸기"

### 창 관리

#### 항상 위 🪟

창을 항상 다른 창 위에 표시:

1. 상단 `📌` 버튼 클릭
2. 버튼이 강조되면 활성화
3. 다시 클릭하면 비활성화

#### 투명도 조절 👻

창 투명도 조절:

1. 사이드바 하단의 "불투명도" 슬라이더
2. 왼쪽으로 이동: 투명
3. 오른쪽으로 이동: 불투명
4. 범위: 30% ~ 100%

## 문제 해결

### 자주 묻는 질문 (FAQ)

#### Q: 메모가 저장되지 않아요
**A**: 타이핑 후 500ms를 기다리세요. 자동 저장됩니다. 좌측 사이드바에서 메모 색상을 확인하세요.

#### Q: 이미지가 표시되지 않아요
**A**:
- 이미지 파일 형식 확인 (PNG, JPG, GIF, BMP)
- `memo_images` 폴더의 파일이 삭제되지 않았는지 확인
- 이미지 크기가 너무 크면 로딩이 느릴 수 있습니다

#### Q: 동영상이 재생되지 않아요
**A**:
- 인터넷 연결 확인
- YouTube/Vimeo URL이 올바른지 확인
- 동영상이 공개 상태인지 확인

#### Q: 표의 셀 크기를 조절할 수 없어요
**A**:
- 셀의 오른쪽 또는 하단 경계선(5픽셀 이내)에 마우스를 올리세요
- 커서가 양방향 화살표로 변경되면 드래그하세요

#### Q: 비밀번호를 잊어버렸어요
**A**:
- 암호화된 메모는 복구할 수 없습니다
- `memos.json` 파일을 직접 편집하여 잠금을 해제할 수 있지만 권장하지 않습니다

#### Q: 프로그램이 느려요
**A**:
- 메모 개수가 많으면 느려질 수 있습니다
- 오래된 메모는 내보내기 후 삭제하세요
- 이미지와 동영상이 많으면 성능에 영향을 줍니다

### 오류 해결

#### 프로그램이 시작되지 않음

```bash
# 필수 라이브러리 재설치
pip install --upgrade customtkinter pillow requests
```

#### JSON 파일 오류

```bash
# 백업 후 파일 삭제
mv memos.json memos.json.backup
mv settings.json settings.json.backup
# 프로그램 재시작 (새 파일 생성됨)
```
